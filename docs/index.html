What is GTTS?

GTTS is a mobile-first platform for truck fueling and toll payments, designed to replace traditional OBU devices and fuel cards.

The solution is built with a future EETS-compatible architecture and focuses on fraud prevention, credit control, and scalability.

MVP Capabilities

  • Fuel stations on interactive map
  • Station-based fueling start
  • Geofence & anti-fraud rules
  • Credit & pre-authorization engine
  • Fuel session lifecycle
  • Hidden TEST MODE admin panel

Technology

Flutter Android iOS Google Maps Riverpod Rule-based Fraud

Roadmap

  • EETS Phase 1–2 compliance
  • Fuel provider integrations (UTA / DKV / E100-like)
  • Backend reconciliation & invoicing
  • Fleet management dashboard
